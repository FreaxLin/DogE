# Doge
Doge是一种模范Java的小型语言，包括编译，加载，虚拟机命令执行和垃圾回收算法。

语法示例如下:
```
Class Test{
    int m = 1;
    int k;
    int compute(){
        int i = 1;
        int j = 1;
        while(i < 10){
            Test t = new Test();
            print(i);
            i = i + j;
        }
        print(i);
        return 0;
    }

}
```
